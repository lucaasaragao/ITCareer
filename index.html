Quem somos ?

O ITCareer é uma plataforma de orientação estudantil com elementos de gamificação, inicialmente voltada para universitários. Com o objetivo de melhorara experiencia acadêmica.

O que fazemos ?

Basicamente tentamos traçar a devida orientação acadêmica com base no perfil apresentado pelo aluno, ou seja, o orientamos a seguir uma trilha de estudos, exercícios e atividades.

O que nos motiva ?

A necessidade de uma orientação no início do curso deve ser levada como regra para nossa plataforma. Ajudar os alunos a encontrar o match perfeito entre skills e carreia é o que motiva a criação do IT Carrer.
